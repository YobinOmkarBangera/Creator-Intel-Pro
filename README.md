# 🛡️ Creator Intelligence Audit (CIA) System
### *Data-Driven Due Diligence for Influencer Marketing*

**Creator Intel Pro** is a high-performance analytics dashboard designed to help Brand Managers and Agencies audit YouTube creators before high-spend sponsorships. It leverages the **YouTube Data API v3** and **Natural Language Processing (NLP)** to provide a 360-degree performance audit.

## 📊 Key Business Metrics
* **Organic Engagement Rate:** Calculated via real-time (Likes + Comments) / View density.
* **Bot Integrity Audit:** Identifies automated spam patterns and repetitive comment structures.
* **Sentiment Polarity Mapping:** A 100-point interaction audit to gauge audience loyalty vs. friction.
* **Performance Benchmarking:** Comparative analysis of the last 30 video uploads.

## 🛠️ Tech Stack
- **Backend:** Python 3.x
- **Data Processing:** Polars (High-speed DataFrame processing)
- **Intelligence:** TextBlob (NLP / Sentiment Analysis)
- **Frontend:** Streamlit (Enterprise Dashboard UI)
- **API:** Google Cloud Console / YouTube Data API v3
